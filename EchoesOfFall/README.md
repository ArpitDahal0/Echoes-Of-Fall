# 🌆 Echoes of Fall

> A brutal vertical rage platformer — Climb the streets




## 🎮 Controls

| Key | Action |
|-----|--------|
| A / ← | Move Left |
| D / → | Move Right |
| Space / ↑ / W | Jump |
| M | Mute |
| P | Pause |

## 🚀 Run the Game


1. Double-click `start.bat`
2. Open Chrome → `http://localhost:8080`

## 🛠️ Tech

HTML5 Canvas · Matter.js · Web Audio API · Vanilla JS · No frameworks
